# Windows-C盘智能安全清理大师 Skill
## 项目简介
本项目专为 **Windows 系统** 打造，后续会迭代开发 macOS 适配版本。仓库提供两套完整资源，分别适配网页端、桌面IDE客户端、CLI终端，用于Windows C盘空间扫描、分级安全清理。
核心逻辑：先完整扫描分区文件并做三级风险划分，安全垃圾给出完整清理方案，高危/系统文件强制人工确认；全程仅依赖Windows原生官方工具，无第三方软件、无高危删除脚本，杜绝误删系统文件蓝屏、丢失数据。

## 核心功能亮点
1. **三重安全隔离**
严格划分系统核心目录黑名单，永久屏蔽关键系统文件；普通缓存垃圾自动梳理清理；休眠文件、系统备份、个人下载文件等高风险项全部交由用户手动确认，杜绝一键误删。
2. **纯官方工具执行**
不推荐第三方清理软件，全部操作基于系统自带磁盘清理、命令行工具，无捆绑、无隐私窃取风险。
3. **标准化结构化输出**
固定六段式报告，每条文件附带路径、占用空间、删除影响，小白也能看懂操作边界。
4. **三端全覆盖**
一套压缩包同时支持桌面IDE、CLI终端；配套独立文件适配网页在线平台，一套资源满足全部使用场景。
5. **轻量化无依赖**
无额外安装程序、无需配置API密钥，导入即可使用。

## 标准六段输出结构
1. 🛑 红线警示：C盘绝对禁止删除的系统核心文件清单
2. ✅ 安全自动清理清单：文件路径、占用空间、清理说明
3. ⚠️ 高危谨慎清单：逐条标注删除风险、操作后果
4. 系统磁盘清理完整新手操作教程，附带安全勾选方案
5. C盘超大占用文件快速查找方法
6. 汇总清单：可直接清理内容 + 需要人工确认内容

## 技能触发关键词

`C盘清理`、`清理C盘`、`C盘满了`、`C盘瘦身`、`释放C盘空间`、`磁盘清理`

# 分渠道完整使用教程（仅Windows系统）
## 第一部分：网页端 Claude.ai 在线平台
使用文件：`Windows-C盘智能安全清理大师.skill`
1. 浏览器打开 `claude.ai`，登录你的Claude账号
2. 左侧侧边栏打开菜单，找到【Skills】技能管理入口
3. 点击上传技能按钮，选中本地 `Windows-C盘智能安全清理大师.skill` 文件确认上传
4. 上传完成后新建对话，输入任意触发词即可自动加载C盘清理全套规则
> 注意：网页端技能储存在云端账号，无法本地修改，如需调整提示词需要重新上传新文件

## 第二部分：桌面IDE客户端（Cursor / Claude Code）
使用文件：`cdisk-clean-wyp.rar`
### 前置解压步骤
1. 右键 `cdisk-clean-wyp.rar` → 选择【全部解压】，得到文件夹 `cdisk-clean`
2. 打开文件夹确认内部存在大写文件名 `SKILL.md`，禁止修改文件名、文件夹名称

### 全局安装（所有项目均可调用，推荐）
#### Cursor 编辑器
1. 打开此电脑，顶部地址栏粘贴路径回车（替换为你自己的Windows用户名）
`C:\Users\你的用户名\.cursor\skills\`
2. 看不到 `.cursor` 文件夹：顶部【查看】选项，勾选「隐藏的项目」
3. 无 `skills` 文件夹则手动新建，命名为全小写 `skills`
4. 将完整解压后的 `cdisk-clean` 文件夹放入 `skills` 目录
5. **完全关闭Cursor全部窗口后重新打开软件**（必须重启才能加载新技能）
6. 调用方式：聊天框输入 `/` 唤起技能列表选择 `cdisk-clean`；直接输入触发词也可自动激活

#### Claude Code 桌面端
1. 地址栏粘贴路径跳转：
`C:\Users\你的用户名\.claude\skills\`
2. 放入解压完成的 `cdisk-clean` 文件夹，彻底关闭客户端重启生效
3. 聊天框输入 `/cdisk-clean` 快速唤起技能

### 局部安装（仅当前项目生效）
在你正在开发的项目根目录，创建文件夹层级：
`.cursor/skills/cdisk-clean/` 或 `.claude/skills/cdisk-clean/`
放入 `SKILL.md`，重启客户端后仅本项目可使用该清理技能

## 第三部分：CLI 终端 / OpenAI Codex 命令行工具
使用文件：`cdisk-clean-wyp.rar`
1. 解压 `cdisk-clean-wyp.rar` 得到 `cdisk-clean` 文件夹，内部保留 `SKILL.md`
2. 找到CLI全局技能存放目录：
`C:\Users\你的用户名\.codex\skills\`
3. 将 `cdisk-clean` 完整文件夹放入 `skills` 目录
4. 关闭所有终端窗口，重新打开CLI终端
5. 使用指令：输入 `/cdisk-clean` 加载技能，或直接输入触发词启动C盘扫描清理分析

# 本地技能常见失效排查（输入`/`看不到技能）
1. 文件夹名包含中文、空格、大写字母 → 必须纯英文小写 `cdisk-clean`
2. 文件名为小写 `skill.md` → 官方强制大写 `SKILL.md`
3. 直接将SKILL.md丢在skills根目录，未嵌套独立技能文件夹
4. 修改文件/文件夹后未彻底关闭软件、终端重启
5. 存放路径存在中文文件夹，建议统一放置系统用户根目录
