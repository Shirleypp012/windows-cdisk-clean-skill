⚡ Quick Start
Requirements
- Windows 10 / Windows 11
- Claude / Cursor / Claude Code / Codex CLI
- No extra environment & dependencies
Installation Guide (Three Ends)
1. Claude Web (Online)
1. Open claude.ai
2. Enter Skills management page
3. Upload Windows-C盘智能安全清理大师.skill
4. Trigger with keywords: C盘清理
2. Cursor / Claude Code (Desktop IDE)
1. Unzip cdisk-clean folder
2. Put into global skill directory:C:\Users\用户名\.cursor\skills\
3. Restart client completely
4. Call via /cdisk-clean or trigger keywords
3. Codex CLI (Terminal)
1. Place skill folder in C:\Users\用户名\.codex\skills\
2. Restart terminal
3. Directly trigger intelligent cleaning analysis
